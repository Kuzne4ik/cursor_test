# WEB Examples for XHE

## Директория примеров работы с веб-функциями XHE

В этой директории содержатся примеры использования классов из пространства имён `XHE\WEB` для работы с браузером, HTTP-запросами, файлами и другими веб-операциями.

## Структура директории

| Файл | Описание |
|------|----------|
| **browser.php** | Примеры управления браузером: открытие страниц, навигация, управление вкладками и окнами |
| **file_manager.php** | Работа с файловой системой: загрузка, скачивание, чтение и запись файлов |
| **http_client.php** | Примеры использования HTTP-клиента: отправка запросов, обработка ответов, работа с cookies |
| **image.php** | Работа с изображениями: создание, редактирование, конвертация форматов |
| **json.php** | Работа с JSON: парсинг, валидация, преобразование данных |
| **mail.php** | Отправка писем через SMTP: создание писем, вложения, HTML-форматирование |
| **screen.php** | Скриншоты экрана: захват всего экрана или отдельных элементов |
| **sound.php** | Работа со звуком: запись, воспроизведение, конвертация аудио |
| **system.php** | Системные функции: работа с реестром, переменными окружения, процессами |
| **text.php** | Работа с текстом: кодирование, декодирование, поиск и замена |
| **video.php** | Работа с видео: создание, конвертация, извлечение кадров |
| **web_proxy.php** | Примеры использования прокси-сервера: настройка, авторизация, ротация IP |

## Инструкции по запуску

1. Убедитесь, что у вас установлен XHE (Extreme Hosting Environment)
2. Скопируйте пример в рабочую директорию вашего XHE проекта
3. Запустите через XHE Host или веб-интерфейс

## Основные классы

### Браузер (WEB::$browser)
- Управление окнами и вкладками
- Навигация по страницам
- Работа с историей посещений

### HTTP-клиент (WEB::$http)
- Отправка GET/POST запросов
- Работа с cookies и сессиями
- Обработка заголовков

### Файловый менеджер (WEB::$file)
- Загрузка и скачивание файлов
- Работа с директориями
- Чтение и запись файлов

### Изображения (WEB::$image)
- Создание и редактирование изображений
- Конвертация форматов
- Работа с текстом на изображениях

### JSON (WEB::$json)
- Парсинг JSON-строк
- Валидация JSON
- Преобразование в массивы

### Почта (WEB::$mail)
- Отправка писем через SMTP
- Работа с вложениями
- HTML-письма

### Скриншоты (WEB::$screen)
- Захват экрана
- Скриншоты элементов
- Сохранение в файл

### Звук (WEB::$sound)
- Запись звука
- Воспроизведение аудио
- Конвертация форматов

### Система (WEB::$system)
- Работа с реестром Windows
- Управление процессами
- Переменные окружения

### Текст (WEB::$text)
- Кодирование/декодирование
- Поиск и замена
- Работа с регулярными выражениями

### Видео (WEB::$video)
- Создание видео
- Извлечение кадров
- Конвертация форматов

### Прокси (WEB::$proxy)
- Настройка прокси-серверов
- Авторизация
- Ротация IP
