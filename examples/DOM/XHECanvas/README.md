# XHECanvas Examples

В этом разделе представлены примеры использования класса XHECanvas для работы с элементами canvas на веб-страницах.

## Описание функций

### draw_image_by_number
Нарисовать картинку с заданным номером на элементе canvas.

**Файл примера:** [`draw_image_by_number.php`](draw_image_by_number.php)

```php
// Нарисовать картинку на первом canvas элементе
$canvas_number = 0;
$image_path = "C:/images/sample.png";
$result = DOM::$canvas->draw_image_by_number($canvas_number, $image_path);
```

## Общие параметры

Функция поддерживает следующие параметры:

- `number` - номер элемента canvas на странице (начиная с 0)
- `path` - путь к файлу с картинкой, которую надо нарисовать на элементе
- `frame` - номер фрейма или строка с номерами фреймов, разделенных двоеточием (по умолчанию "-1")

## Примечания

- Все примеры используют стандартную инициализацию XHE API
- В конце каждого примера вызывается `WINDOW::$app->quit()` для завершения работы
