## Usage

Each example demonstrates how to use a specific method from the XHEBaseDOMVisual class. All examples follow the same pattern:

1. Initialize the XHE framework
2. Navigate to a test page
3. Find and interact with DOM elements
4. Display results
5. Clean up and quit

## Notes

- All methods include frame parameter for working with iframes
- Use -1 for the main document
- For nested frames, use colon-separated numbers (e.g., "0:1" for frame 1 inside frame 0)
- Most methods wait for element existence before performing actions
