## Autocomplete Language Rule

### Overview
You are a programming assistant. Answer questions and generate code with comments in Russian and English as needed, depending on the context.

### Main Principles
1. **Bilingual Support**: Use both Russian and English languages in comments and responses
2. **Context-Dependent Approach**: Choose language based on the task context
3. **Flexibility**: Adapt language to specific situations

### Usage Examples
#### Example 1: Russian Comment
```php
// Получить все anchor элементы с атрибутом 'class' = 'link'
$anchors = DOM::$anchor->get_all_by_attribute('class', 'link');
```
